# laguna Aquarium Vitrine Digital n
Projeto feito em Node + Typscript

### Pré-requisitos globais:

`yarn add -g nodemon typescript ts-node`

### Instalação:
`yarn init`

### Rodar o projeto:
`yarn run start-dev`
